# Java Processing Physics Engine
A 2D physics engine made with Processing, prioritizing modularity and accuracy 
to classical physics. To be used to make astronomical models as well as classical 
physics models and video games.
The modularity allows theoretical physics models to be created through the 
isolation of systems of particles and the ability to change universal constants.

## Current Models/Features
Current supported models:
- 0D particles

Current supported features:
- 2D kinematics
- Forces
- Static downwards gravity
- Dynamic planetary gravity

## Planned Models/Features
Upcoming models:
- 1D segments
- 2D shapes including polygons built by segments or simple shapes

Upcoming features:
- Conservation of momentum
- Elastic/inelastic collisions

## Proposed Models/Features
Models and features that have been considered but are not yet planned:
- Electrostatics
- Magnetism
